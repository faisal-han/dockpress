# DockPress

`dockpress` is a security-first CLI generator for Express.js. It helps you scaffold a production-ready Express.js application with a hardened Docker configuration, following industry best practices to minimize security risks.

## Features
- **Security-First**: Generates Dockerfiles with non-root user configurations to mitigate container breakout risks.
- **Production-Ready**: Comes with a clean, modular structure optimized for deployment.
- **Easy Scaffolding**: Setup your Express.js backend in seconds.

## Installation
Install it globally via npm:

```bash
npm install -g dockpress
